Processing Flow

API 호출부터 Instagram 게시까지의 상세 시퀀스

API

POST /api/characters

캐릭터 컨셉, 톤, 주제를 입력받아 생성 요청

Input: { concept, tone, topics[] }
→ characterPrompt.builder.js → Claude API 호출
AI

Claude API → Character Profile JSON

페르소나, 외형, IG 프로필, 음성 가이드, 브랜드 안전 규칙 생성

Output: { name, age(23+), personality[], visualDescription,
  instagramProfile, voiceGuidelines, brandSafety }
SAFETY

Character Validation

Zod 스키마 검증 + 28개 Blocked Terms 전수 스캔 + Age Gate (23세 미만 거부)

Pass → DB 저장 → Step 4
Fail → 재생성 (최대 2회) → 실패 시 422 반환
API

POST /api/characters/:id/images/generate

캐릭터 프로필 기반 이미지 4장 생성 요청

imagePrompt.builder.js: persona.visualDescription → 4가지 Scene Variation
- scene_0: cafe | scene_1: street | scene_2: studio (master) | scene_3: rooftop
+ SAFETY_NEGATIVE_PROMPT 자동 주입 + SAFETY_POSITIVE_ENFORCEMENT 접미
PROVIDER

Image Provider → 4 Candidates

Replicate(FLUX) 또는 fal.ai에 4회 병렬 호출, 개별 재시도 2회

Promise.allSettled() → 성공한 것만 DB 저장
전부 실패 시 Job status → failed, 502 반환
DB

Master Image 선정

scene_2 (스튜디오 headshot) 우선 → 트랜잭션으로 기존 master 강등 + 새 master 지정

image_assets.status: 'candidate' → 'master' (1개)
수동 변경: PUT /api/characters/:id/images/:imageId/master
API

POST /api/characters/:id/videos/generate

Master Image 기반 Instagram Reels 영상 생성

videoPrompt.builder.js: Style Preset + Motion Prompt 조립
Styles: slow_motion | natural | dynamic | cinematic | loop
Output: 1080x1920, 3~10초, 9:16 비율
PROVIDER

Video Provider → Submit + Poll

Runway/Kling/Minimax에 비동기 제출, 5초 간격 폴링 (최대 10분)

submit() → providerJobId 수신
poll() → queued → processing → completed
실패 시 재시도 2회 (5초, 10초 대기)
API

POST /api/contents

이미지/영상을 게시 단위 콘텐츠로 묶기 + 캡션 자동 생성

captionPrompt.builder.js: persona.voiceGuidelines 기반 캡션 생성
Claude API → { caption, hashtags(10~20개), callToAction, altText }
해시태그: 스팸 태그 자동 제거 + 30개 제한 + NSFW 태그 필터
REVIEW

approve → schedule → publish

검수 승인 → 예약 게시 → Instagram Graph API로 실제 게시

POST /contents/:id/approve → status: approved
POST /contents/:id/schedule → publish_jobs 생성
POST /contents/:id/publish-now → 즉시 게시 (approve+schedule+execute)

Content Policy

Safety & Moderation

3단계 안전 장치: 프롬프트 가드 + 금지어 스캔 + 해시태그 필터

✖ 절대 차단

  • 미성년 암시 (child, teen, school uniform, loli)
  • 누드/노출 (nude, naked, lingerie, see-through)
  • 성적 표현 (sexual, fetish, erotic, seductive)
  • 폭력/유해 (violence, weapon, blood, drug)
  • 스팸 해시태그 (f4f, l4l, followback)
  • IG 제한 태그 (nsfw, onlyfans, boudoir)

✔ 허용 기준

  • 캐릭터 나이: 23세 이상만 허용
  • 의상: 일상복, 비즈니스 캐주얼
  • 포즈: 자연스러운 일상 포즈
  • 표정: 밝고 건전한 표정
  • 배경: 카페, 거리, 스튜디오, 자연
  • 톤: 친근하고 브랜드 세이프

⚠ 자동 가드

  • 모든 이미지 prompt에 negative prompt 강제 주입
  • 모든 prompt에 "clearly adult, age 25+, SFW" 접미
  • 모든 영상 prompt에 "fully clothed throughout" 주입
  • 해시태그 30개 자동 제한 + 중복 제거
  • 캡션 2200자 제한 (Instagram 규격)
  • 캐릭터 생성 실패 시 최대 2회 자동 재생성
